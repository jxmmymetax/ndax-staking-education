# How to Stake Solana on NDAX

This project is a short educational walkthrough explaining what staking is and how a user can navigate the Solana staking process on NDAX.

## What the video covers

- What cryptocurrency staking means
- How staking helps support a Proof-of-Stake network
- How to locate the staking section on NDAX
- How to review estimated rewards and unstaking terms
- Where a user would enter an amount and confirm the request

## Video

[View the staking walkthrough](video/how-to-stake-solana-on-ndax.mp4)

## Key takeaway

Staking allows eligible cryptocurrency holders to delegate their assets to support a blockchain network and potentially receive rewards.

Reward rates are variable, cryptocurrency prices can rise or fall, and unstaking may involve waiting periods.
